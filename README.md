##  Project Overview
This repository is created as part of the **CSE0001 – Digital Literacy** course project.  
The project is based on the scenario of being a **Student Digital Ambassador**, where the goal is to help peers understand digital tools, professional online presence, collaboration platforms, email etiquette, and cyber safety.

This project includes **5 tasks**, each related to one module of the course.  
All tasks are organized into separate folders with proper documentation, reflections, and supporting files.
